# TowerOfHanoiJS #
JavaScript code to create Tower of Hanoi game.

Version: 0.1

This code itself is not a jQuery plugin, but this code needs jQuery and jQuery UI
to run. Make sure jQuery and jQuery UI are loaded first before create 
the Game instance.
